# lux2efficacy

## lux2eff.py
给定单色光的波长和照度（lux），得到辐照（irradiance）。


## ledCal.py
给定lux，得到led灯的辐照。
拿了一个LED灯光谱，假设不同lux的LED灯光谱等比缩放。
